mmj.mmio

Class Statementizer

  • java.lang.Object
    • mmj.mmio.Statementizer

  • public class Statementizer
extends java.lang.Object
    Load input MetaMath tokens for a single Metamath statement into a "work" structure, SrcStmt.

    (This is actually a somewhat inefficient intermediate step on the way to loading the source into memory, but is done this way to simplify the main code and to improve testability.)

      • bypassErrorStmt

        public void bypassErrorStmt()
                     throws java.io.IOException
        Attempts to bypass the current statement in the input Metamath file.

        Bypass is intended to give the next next invocation of getStmt() a good chance of finding a complete statement to parse, thus avoiding multiple error messages caused by a single error.

        Throws:
        java.io.IOException - if I/O error

      • setStmtNbr

        public void setStmtNbr(int s)
                throws java.lang.IllegalArgumentException
        Sets statement number of the last statement parsed.

        This may come in handy in conjunction with included files -- construct a new Statementizer for the included file, set the stmtNbr, process, then continue processing with the old Statementizer... after setting its stmtNbr. Or...just use the setTokenizer method to switch Tokenizers! Easy.

        Parameters:
        s - Metamath statement (sequence) number.
        Throws:
        java.lang.IllegalArgumentException - if input statement number is less than zero.
