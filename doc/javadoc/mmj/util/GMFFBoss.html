      • getGMFFManager

        public GMFFManager getGMFFManager()
        Fetch a gmffManagerobject, building it if necessary from previously input RunParms.

        NOTE: The returned gmffManageris "ready to go" but may not have been "initialized", which means gmffManagervalidation, etc. The reason that gmffManageris not initialized here is that a previous attempt to "initialize" may have failed due to gmffManagererrors, so to re-do it here would result in doubled-up error messages. The Initialize gmffManagerRunParm Command should be used prior to PrintSyntaxDetails if a "load and print syntax" is desired.

        Returns:
        gmffManagerobject, ready to go.
