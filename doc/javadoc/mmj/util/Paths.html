      • buildMMJ2FilePath

        public java.io.File buildMMJ2FilePath(java.lang.String pathName)
        Builds a File object relative to the mmj2Path.

        If the input fileName designates an absolute path then the output File is not relative to mmj2Path or the Current Path.

        Otherwise, if mmj2Path is null then the output File is relative to the Current Path.

        Parameters:
        pathName - name of file or directory.
        Returns:
        File object relative to mmj2Path unless input pathName is absolute or mmj2Path is null, in which case the output File is relative to the current path.
