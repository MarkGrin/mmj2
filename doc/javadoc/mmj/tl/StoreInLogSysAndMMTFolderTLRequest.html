Class StoreInLogSysAndMMTFolderTLRequest

  • java.lang.Object
    • mmj.tl.TLRequest
      • mmj.tl.StoreInLogSysAndMMTFolderTLRequest

  • public class StoreInLogSysAndMMTFolderTLRequest
extends TLRequest
    StoreInLogSysAndMMTFolderTLRequest implements a user request for a TheoremLoader operation.

    This class is basically just a name which is used by the ProofAsstGUI to identify the user's request, and a call to implement the request.

    The input ProofWorksheet must be already successfully unified, or else a TheoremLoaderException is thrown. The ProofWorksheet is converted to Metamath .mm format and stored in the MMT Folder, and then is loaded into the LogicalSystem currently in memory.
