      • loadContentsToString

        public java.lang.String loadContentsToString()
                                      throws GMFFException,
                                             GMFFFileNotFoundException
        Returns string containing the entire file contents.

        A BufferedReader is created using the specified buffer size and a StringBuilder is loaded with repeated reads until end of file reached. Then the StringBuilder is converted to a String and returned to the called.

        Obviously, this function has a problem if asked to read a humongous file. (We should test that :-)

        Returns:
        String containing the entire file contents.
        Throws:
        GMFFFileNotFoundException - if file not found
        GMFFException - if other error, such as IOException or SecurityException.
