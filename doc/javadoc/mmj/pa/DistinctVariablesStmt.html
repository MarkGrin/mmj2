Class DistinctVariablesStmt

  • java.lang.Object
    • mmj.pa.ProofWorkStmt
      • mmj.pa.DistinctVariablesStmt

  • public class DistinctVariablesStmt
extends ProofWorkStmt
    DistinctVariablesStmt represents a single Metamath $d statement in a ProofWorksheet.

    A proof may contain 0 -> n DistinctVariablesStmt's. They are used during Unification as *additions* to whatever $d statements are contained in the database for the theorem being proved.
