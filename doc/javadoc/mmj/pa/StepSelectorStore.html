Class StepSelectorStore

  • java.lang.Object
    • mmj.pa.StepSelectorStore

  • public class StepSelectorStore
extends java.lang.Object
    StepSelectorStore accumulates StepSelector results.

    The design is rudimentary because it does not sort or select which results to keep -- at this time.

    Its main purpose is to provide a simple data store for StepSelectorSearch.

      • createStepSelectorResults

        public StepSelectorResults createStepSelectorResults(java.lang.String step,
                                            boolean storeOverflow)
        Creates a StepSelectorResults object using the contents of the data store.

        An extra item is added to the StepSelectorResults. If the store contains one extra result (beyond the specified StepSelectorMaxResults parameter), then "***MORE***" is output at the end of the StepSelectorResults selection array along with a null Assrt. Otherwise it outputs "***END***". The purpose it to inform the user that rerunning the StepSelector may or may not be useful.

        Parameters:
        step - Step String corresponding to the ProofWorksheet step (number) for which the StepSelector was run.
        storeOverflow - indicates whether or not additional results were available but not added to the store because it was full.
        Returns:
        StepSelectorResults object for display on the StepSelectorDialog.
