Class FooterStmt

  • java.lang.Object
    • mmj.pa.ProofWorkStmt
      • mmj.pa.FooterStmt

  • public class FooterStmt
extends ProofWorkStmt
    FooterStmt is simply the "$)" in column 1 of the last line of a proof.

    Note: the purpose of FooterStmt is to provide a terminator so that multiple proofs can be stored in a file (and this is done for batch volume testing.)
